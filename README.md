# Hacktiv8 Final Project 4

## Installation
Requires Golang and MySQL

Edit .env first to sql configuration and server port

Create a "final4" database

```
$ git clone https://github.com/verssache/hacktiv8-final4
$ cd hacktiv8-final4
$ go run main.go
```
